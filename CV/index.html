    Golds gym

    Golds gym is a fitness platform that uses React, React Router and Material UI for its user interface and user experience. It makes use of hooks like useState and useEffect to manage state and side effects in the components. To retrieve workout data, the site makes use of the Fetch API to make calls to the RapidAPI, where all the available exercises are stored.
